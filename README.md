A conversational German language learning chatbot that runs locally on your machine using Ollama and LLaMA3.

## About
This chatbot helps users practice conversational German. It understands beginner level German and responds with English translations to help you learn.

## Tech Stack
- Python
- Ollama
- LLaMA3 (runs locally, completely free)

## How to Run
1. Install [Ollama](https://ollama.com) and pull the LLaMA3 model
2. Clone this repo
3. Create a virtual environment and activate it
4. Install dependencies with `pip install ollama`
5. Run with `python3 chatbox.py`

## Features
- Runs 100% locally, no API costs
- Understands broken/beginner German
- Responds in German with English translations
- Conversational and beginner friendly
